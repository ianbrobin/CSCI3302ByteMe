CSCI 3302 - Intro to Robotics

Team Name: ByteMe

Austin Albert - Chandler Garthwaite - Connor Thompson - Ian Brobin
This is our final project implementation for CSCI 3302. In this project, we have a few parts:
  • We have this website, which acts as our client, allowing you to input Tic-Tac-Toe moves and reset the game if needed.

  • We have a local TurtleSim running which acts as a computer player against you. This can be viewed in your terminal. The turtle uses alpha-beta pruning to make optimal moves, so it will never lose. Only tie or win.

  • Lastly, we have a ROSBridge set-up and running which facilitates communication between the client and our server (turtle). This allows messages to be passed between our client/server using ROS's Publish/Subscribe model.

All code for this project can be found here.
We hope you enjoy :)
